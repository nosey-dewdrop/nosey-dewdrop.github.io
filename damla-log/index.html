🌱

versions

plan releases with feature lists. track status from planning to shipped. know exactly what goes into each update.

how it works

from terminal to pocket.

1

code with claude

work on your projects with claude code in the terminal. tasks and issues get logged automatically.

2

auto-sync

url scheme sends tasks, ideas, and issues directly to the app. no copy-paste, no context switching.

3

check your phone

icloud keeps everything synced. open the app anywhere and see exactly where every project stands.

4

ship it

plan versions, track features, mark them shipped. your entire dev lifecycle in one place.
